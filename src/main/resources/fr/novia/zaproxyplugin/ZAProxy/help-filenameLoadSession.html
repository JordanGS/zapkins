Permet de choisir la session à charger dans la liste fournie.
Si aucune session n'est proposée c'est qu'elle n'a pas été sauvegardée au moins une fois. Aucune session ne sera donc chargée. Une fois la session est chargée, il n'est pas nécessaire de la sauvegarder. Toutes les actions menées seront enregistrées dans la session chargée.
Charger une session permet l'enregistrement des actions potentielles suivantes :
URLS à inclure et à exclure du contexte
Spidering
Ajax spidering
Scan
Sauvegarder le rapport
Pour enregistrer un nouveau mode d'authentification, il est nécessaire de sauvegarder une nouvelle session, la précédente sera conservée.
En effet, on aurait pu mener ces actions, mais pour des raisons de simplicité cela n'est pas possible avec cette version du plugin.
