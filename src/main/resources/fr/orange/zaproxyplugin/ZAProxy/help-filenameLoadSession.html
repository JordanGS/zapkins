Permet de choisir la session à charger dans la liste fournie.
Si aucune session n'est proposée, c'est qu'elle n'a pas été sauvegardée au moins une fois.
Aucune session ne sera donc chargée. Une fois la session est chargée, il n'est pas nécessaire de la sauvegarder. Toutes les actions menées seront enregistrées dans la session chargée.
Charger une session permet de récupérer les éléments suivants :
-URLS à inclure et à exclure du contexte
-Résultats du spidering et de l'Ajax Spidering
-Résultats du scan
-Mode et paramètres de l'authentification (basé sur le formulaire, basé sur le script d'authentification)
Pour enregistrer un nouveau mode d'authentification, il est nécessaire de sauvegarder une nouvelle session, la précédente sera conservée.
En effet, on aurait pu mener ces actions, mais pour des raisons de simplicité cela n'est pas possible avec cette version du plugin.
