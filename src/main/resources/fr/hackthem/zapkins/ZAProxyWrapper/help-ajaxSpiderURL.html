AJAX Spider permet à ZAP d'identifier des pages supplémentaires en évaluant le code javascript (AJAX) résidant sur les pages du site ciblé.
Il peut être combiné avec le "Spider" standard pour des meilleurs résultats.
Dans ce mode, si l'audit est mené d'une manié authentifiée alors les pages du site ciblé sont visités de manière authentifiée.
